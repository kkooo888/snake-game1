# 网页版贪吃蛇游戏

这是一个使用 HTML5 Canvas 和 JavaScript 开发的经典贪吃蛇游戏。

## 游戏特点

- 简洁美观的界面设计
- 流畅的游戏体验
- 实时分数显示
- 支持暂停/继续功能

## 如何玩

1. 点击"开始游戏"按钮开始
2. 使用键盘方向键控制蛇的移动
3. 吃到红色食物可以得分
4. 撞到墙壁或自己会导致游戏结束

## 在线试玩

访问 [贪吃蛇游戏](https://[您的GitHub用户名].github.io/snake-game/) 即可在线体验游戏。

## 本地运行

1. 克隆此仓库
2. 在浏览器中打开 `index.html` 文件

## 技术栈

- HTML5
- CSS3
- JavaScript
- Canvas API
